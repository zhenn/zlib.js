next()

匹配所匹配元素中第一个元素的下一个兄弟元素节点（过滤非元素节点）

参数

无

返回值

zlib

用法演示

查找id="obj"元素的下一个兄弟元素节点

HTML 代码

<ul id="obj">
　　 <li>zhenn</li>
　　 <li>do you known the eventListener?</li>
　　 <li>let me tell you </li>
　　 <li>success</li>
</ul>

我是文本节点，不是元素节点！
<div class="other">object</div>

zLib 代码

$("#obj").next();

结果

<div class="other">object</div>
