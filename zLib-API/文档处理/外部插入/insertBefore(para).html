insertBefore(para)

把所匹配元素插入到目标元素的第一个元素之前，所匹配元素可以是已经存在于document的对象，也可以是create的新对象（不是dom元素）

参数

para(zlib)：必须是$取得的已经存在于document中的对象

返回值

zlib，所匹配对象

用法演示

在id="obj"元素之前插入一个div，并设置innerHTML="insertBefore"

HTML 代码

<ul id="obj">
　　 <li>zhenn</li>
　　 <li>do you known the eventListener?</li>
　　 <li class="child">let me tell you </li>
　　 <li>success</li>
</ul>

我是文本节点，不是元素节点！
<div class="other">object</div>

zLib 代码

$("*div").insertBefore($("#obj")).html("insertBefore");

结果

<div>insertBefore</div>

<ul id="obj">
　　<li>zhenn</li>
　　<li>do you known the eventListener?</li>
　　<li class="child">let me tell you</li>
　　<li>success</li>
</ul>
