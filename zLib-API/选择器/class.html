.class

根据给定的class匹配元素。

参数

class(String)：一个元素可以有多个类，只要有一个符合就能被匹配到。

返回值

zlib

用法演示

查找class为"obj"的元素

HTML 代码

<div class="obj">object1</div>

<div class="other">object2</div>

<div class="obj" >object3</div>

<div class="obj other">object4</div>

zLib 代码

$(".obj")

结果

[<div class="obj">object1</div>,<div class="obj">object3</div>,<div class="obj other">object4</div>]
