html([val])

给所匹配元素赋innerHTML值，或者返回所匹配元素中第一个元素的innerHTML值

参数

val(String)：可选参数，用于设定innerHTML的值

返回值

1，传参，则为赋值运算，返回zlib
2，无参，返回String，所匹配元素集中第一个元素的innerHTML 值

用法演示

给id="obj"下所有li元素赋innerHTML值

HTML 代码

<ul id="obj"><li>1</li><li>2</li><li>3</li><li>4</li></ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").html("I'm new content!")

结果

[<ul id="obj"><li>I'm new content!</li><li>I'm new content!</li><li>I'm new content!</li><li>I'm new content!</li></ul>]

获得id="obj"下第一个li元素innerHTML值

HTML 代码

<ul id="obj"><li>1</li><li>2</li><li>3</li><li>4</li></ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").html()

结果

"1"
