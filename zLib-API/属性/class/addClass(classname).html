addClass(classname)

给所匹配元素添加className

参数

classname(String)：要添加的className

返回值

zlib

用法演示

给id="obj"下的所有li元素添加className->newClass

HTML 代码

<ul id="obj"><li class="oldClass zhenn" >1</li><li>2</li><li>3</li><li>4</li></ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").addClass("newClass");

结果

<ul id="obj">

　　<li class="oldClass zhenn newClass">1</li>

　　<li class="newClass">2</li>

　　<li class="newClass">3</li>

　　<li class="newClass">4</li>

</ul>
