attr(name,[value])

获得所匹配元素中第一个元素的属性name的值，或者给所匹配元素赋单个属性值

参数

name(String)：所要操作的属性名
value(String)：可选参数，所要操作的属性值

返回值

zlib

用法演示

给id="obj"下的所有li元素增加属性title="newTitle"

HTML 代码

<ul id="obj">
　　 <li>1</li>
　　 <li>2</li>
　　 <li>3</li>
　　 <li>4</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").attr("title","newTitle");

结果

<ul id="obj">
　　 <li title="newTitle">1</li>
　　 <li title="newTitle">2</li>
　　 <li title="newTitle">3</li>
　　 <li title="newTitle">4</li>
</ul>
