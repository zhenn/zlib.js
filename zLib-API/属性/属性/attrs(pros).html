attrs(pros)

给所匹配元素赋一组属性值

参数

pros(Object)：所要操作的一组属性,不包括class（class是JavaScript关键字）,如要添加className,可使用函数addClass或者attr

返回值

zlib

用法演示

给id="obj"下的所有li元素增加一组属性{name:"newName",title:"newTitle"}

HTML 代码

<ul id="obj">
　　 <li>1</li>
　　 <li>2</li>
　　 <li>3</li>
　　 <li>4</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").attrs({name:"newName",title:"newTitle"});

结果

<ul id="obj">
　　 <li title="newTitle" name="newName">1</li>
　　 <li title="newTitle" name="newName">2</li>
　　 <li title="newTitle" name="newName">3</li>
　　 <li title="newTitle" name="newName">4</li>
</ul>
