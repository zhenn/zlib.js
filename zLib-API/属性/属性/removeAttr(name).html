removeAttr(name)

删除所匹配元素单个或一组属性

参数

name(String or Array)：所要操作的一组属性

返回值

zlib

用法演示

删除id="obj"下的所有li元素属性name，title

HTML 代码

<ul id="obj">
　　 <li title="newTitle" name="newName">1</li>
　　 <li title="newTitle" name="newName">2</li>
　　 <li title="newTitle" name="newName">3</li>
　　 <li title="newTitle" name="newName">4</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").removeAttr(["name","title"]);

结果

<ul id="obj">
　　 <li>1</li>
　　 <li>2</li>
　　 <li>3</li>
　　 <li>4</li>
</ul>

删除id="obj"元素的title

HTML 代码

<ul id="obj" title="parentTitle" >
　　 <li>1</li>
　　 <li>2</li>
　　 <li>3</li>
　　 <li>4</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj").removeAttr("title");

结果

<ul id="obj">
　　 <li>1</li>
　　 <li>2</li>
　　 <li>3</li>
　　 <li>4</li>
</ul>
