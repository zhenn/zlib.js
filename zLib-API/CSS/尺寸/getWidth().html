getWidth()

获取所匹配元素中第一个元素的潜在宽度，无论是元素隐藏还是没有设置CSS宽度，所以并不等同于getCss("width")和domElements.offsetWidth（仅在元素不隐藏的情况下生效）

参数

无

返回值

Number，宽度

用法演示

获取id="obj"元素的宽度

HTML 代码

<div style="width:800px;">

　　<ul id="obj" style="display:none;" >
　　　　<li sytle="height:20px;">1</li>
　　　　<li sytle="height:20px;">2</li>
　　　　<li sytle="height:20px;">3</li>
　　　　<li sytle="height:20px;">4</li>
　　</ul>

</div>

<ul><li>other</li></ul>

zLib 代码

$("#obj").getWidth();

结果

800
