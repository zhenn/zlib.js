getHeight()

获取所匹配元素中第一个元素的潜在高度，无论是元素隐藏还是没有设置CSS高度，所以并不等同于getCss("height")和domElements.offsetHeight（仅在元素不隐藏的情况下生效）

参数

无

返回值

Number，高度

用法演示

获取id="obj"元素的高度

HTML 代码

<ul id="obj" style="display:none;" >
　　 <li sytle="height:20px;">1</li>
　　 <li sytle="height:20px;">2</li>
　　 <li sytle="height:20px;">3</li>
　　 <li sytle="height:20px;">4</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj").getHeight();

结果

80
