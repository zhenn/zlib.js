gapsY()

获取所匹配元素距窗口最顶端的偏移量（包括由于滚动条隐藏的部分），仅对非display:none的元素有效

参数

无

返回值

Number

用法演示

获取id="obj"元素距窗口最顶端的偏移量（包括由于滚动条隐藏的部分）

HTML 代码

<ul id="obj" style="margin:200px 0 0 100px;" >
　　 <li>1</li>
　　 <li>2</li>
　　 <li>3</li>
　　 <li>4</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj").gapsY();

结果

200
