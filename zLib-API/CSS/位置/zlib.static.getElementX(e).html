zlib.static.getElementX(e)

获得鼠标相对于当前元素(事件对象"e"的属性target)的水平偏移量

参数

e：事件对象

返回值

Number

用法演示

获得鼠标相对于当前元素(事件对象"e"的属性target)的水平和垂直偏移量，并且把实时的显示在window的状态栏里

HTML 代码

无

zLib 代码

$(document).bindEvent("mousemove",function(e){
　　window.status = zlib.static.getElementX(e) + " " + zlib.static.getElementY(e);
});

结果

在状态栏实时的显示鼠标相对于目标元素的水平和垂直方向的偏移量。
