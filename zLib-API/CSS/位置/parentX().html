parentX()

获取所匹配元素相对于父级元素的偏移量(水平方向)，仅对非display:none的元素有效

参数

无

返回值

Number

用法演示

获取id="obj"元素距其父节点元素的偏移量（水平方向）

HTML 代码

<div style="margin:100px 0 0 100px;">
　　<ul id="obj" style="margin:20px 0 0 10px; float:left;">

　　　　　<li>1</li>

　　　　　<li>2</li>
　　　　　<li>3</li>
　　　　　<li>4</li>
　　</ul>
</div>

<ul><li>other</li></ul>

zLib 代码

$("#obj").parentX();

结果

10
