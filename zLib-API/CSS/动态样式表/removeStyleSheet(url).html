zlib.static.removeStyleSheet(url)

删除已经载入的外联样式表stylesheet

参数

url(String)：要删除样式表的url地址

返回值

无

用法演示

动态删除一个样式表print.css

HTML 代码

<link href="style/print.css" rel="stylesheet" type="text/css" media="print" />

<link href="style/global.css" rel="stylesheet" type="text/css" media="screen" />

zLib 代码

zlib.static.removeStyleSheet("style/print.css");

结果

<link href="style/global.css" rel="stylesheet" type="text/css" media="screen" />
