setStyle(property,value)

设置所匹配元素的单个css样式

参数

property(String)：要设置的css属性名
value(String)：要设置的css属性值

返回值

zlib

用法演示

设置id="obj"下所有li元素的样式为color:red;

HTML 代码

<ul id="obj">
　　 <li>1</li>
　　 <li>2</li>
　　 <li>3</li>
　　 <li>4</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").setStyle("color","red");

结果

<ul id="obj">
　　 <li style="color:red;">1</li>
　　 <li style="color:red;">2</li>
　　 <li style="color:red;">3</li>
　　 <li style="color:red;">4</li>
</ul>
