setCss(styles)

设置所匹配元素的一组css样式

参数

styles(Object)：要设置的css样式，如{color:"#f00",height:"100px"}

返回值

oldStyle(Object)，所匹配元素未运行setCss之前的一组css样式{}

用法演示

设置id="obj"下所有li元素的样式为{color:"red",height:"100px"}

HTML 代码

<ul id="obj">
　　 <li>1</li>
　　 <li>2</li>
　　 <li>3</li>
　　 <li>4</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").setCss({color:"red",height:"100px"});

结果

<ul id="obj">
　　 <li style="color:red; height:100px;">1</li>
　　 <li style="color:red; height:100px;">2</li>
　　 <li style="color:red; height:100px;">3</li>
　　 <li style="color:red; height:100px;">4</li>
</ul>
