bindEvent(eventType,callback)

为所匹配元素增加事件监听(绑定事件)

参数

eventType(String)：绑定的事件类型，如"click","load","mouseover","keydown"...
callback(Function)：响应事件的函数

返回值

zlib

用法演示

给id="obj"元素下所有li元素增加事件监听

HTML 代码

<ul id="obj">

　　　<li>zhenn</li>

　　　<li>do you known the eventListener?</li>
　　　<li>let me tell you </li>
　　　<li>success</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").bindEvent("click",function(){

　　$(this).html("test");

});

结果

每点击一个li，将其li的innerHTML值设置为"test";
