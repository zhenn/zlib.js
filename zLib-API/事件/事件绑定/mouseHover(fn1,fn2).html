mouseHover(fn1,fn2)

复合动作mouseover,mouserout的处理程序(事件切换)，模仿悬停事件的方法，自动判断isMouseLeaveOrEnter

参数

fn1(Function)，mouseover所触发的处理程序
fn2(Function)，mouseout触发的处理程序

返回值

无

用法演示

给id="obj"元素下所有li元素绑定复合事件(mouseHover)

HTML 代码

<ul id="obj">

　　　<li>zhenn</li>

　　　<li>do you known the eventListener?</li>
　　　<li>let me tell you </li>
　　　<li>success</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").mouseHover(
　　function(){
　　　　alert("enter");
　　},
　　function(){
　　　　alert("leave");
　　}
);

结果

鼠标移到每个li元素，弹出提示信息"enter"，从每个li元素上移开时,弹出提示信息"leave"
