zlib.static.preventDefault(e)

什么是事件的默认动作？例如，默认情况下点击一个<a>元素,页面会跳转到该元素href属性指定的页面。另外在提交表单的时候，要在客户端判断信息是否属实，是否漏填信息，如果符合上面任一种的条件，那么我们需要，在客户端输出提示信息，而并不会提交至服务器处理，这时候就需要阻止默认事件的发生！

参数

e：事件对象

返回值

无

用法演示

点击超链接，阻止跳转页面

HTML 代码

<ul id="obj">

　　　<li><a href="http://www.men-ideal.com">zhenn</a></li>

　　　<li>do you known the eventListener?</li>
　　　<li>let me tell you </li>
　　　<li>success</li>
</ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").bindEvent("click",function(e){

　　alert($(this).html());
　　zlib.static.preventDefault(e);
});

结果

点击第一个li元素，弹出提示信息，但是不会跳转页面！
