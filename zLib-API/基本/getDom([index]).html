getDom([index])

返回所匹配的dom元素或者dom元素集

参数

index(Number)：可选参数，索引值从 0 开始

返回值

dom元素或者dom元素集

用法演示

查找id="obj"下的第三个li元素，并且设置它的innerHTML。

HTML 代码

<ul id="obj"><li>1</li><li>2</li><li>3</li><li>4</li></ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").getDom(2).innerHTML = "zhenn";

or

$("#obj","li").getDom()[2].innerHTML = "zhenn";

 

很显然，$("#obj","li").getDom(2).html("zhenn")，会提示语法错误，因为$("#obj","li").getDom(2)返回的是dom元素,而不是zlib

结果

<ul id="obj"><li>1</li><li>2</li><li>zhenn</li><li>4</li></ul>
