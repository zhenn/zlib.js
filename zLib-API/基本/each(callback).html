each(callback)

迭代程序，遍历对象中每一个dom元素都执行一次回调函数

参数

callback(Function)：回调函数

返回值

zlib

用法演示

迭代id="obj"下的所有li元素，并且设置它们的innerHTML。注意:此处 el 指代的是 DOM 对象而非 zlib 对象

HTML 代码

<ul id="obj"><li>1</li><li>2</li><li>3</li><li>4</li></ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").each(function(el){

　　el.innerHTML = "zhenn";

});

or

$("#obj","li").each(function(el){

　　$(el).html("zhenn");

});

or

$("#obj","li").html("zhenn");

结果

<ul id="obj"><li>zhenn</li><li>zhenn</li><li>zhenn</li><li>zhenn</li></ul>
