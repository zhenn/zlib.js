getClass(index)

从实例对象中返回一个和索引值相匹配的实例对象，以便调用zlib类中的方法

参数

index(Number)：索引值从 0 开始

返回值

zlib

用法演示

查找id="obj"下的第三个li元素，并且设置它的innerHTML。

HTML 代码

<ul id="obj"><li>1</li><li>2</li><li>3</li><li>4</li></ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").getClass(2).html("zhenn");

结果

<ul id="obj"><li>1</li><li>2</li><li>zhenn</li><li>4</li></ul>
