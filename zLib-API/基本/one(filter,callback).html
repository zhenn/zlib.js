one(filter,callback)

有条件寻找一个dom对象来执行回调函数

参数

filter(Number or String)：查找dom元素的两种格式（索引值从 0 开始）

callback(Function)：回调函数

返回值

zlib

用法演示

查找id="obj"下的第三个li元素，并且设置它的innerHTML。注意:此处 el 指代的是 DOM 对象而非 zlib 对象

HTML 代码

<ul id="obj"><li>1</li><li>2</li><li>3</li><li>4</li></ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").one(2,function(el){

　　el.innerHTML = "zhenn";

});

or

$("#obj","li").one(2,function(el){

　　$(el).html("zhenn");

});

结果

<ul id="obj"><li>1</li><li>2</li><li>zhenn</li><li>4</li></ul>

查找id="obj"下的title="new"的li元素，并且设置它的innerHTML。注意:此处 el 指代的是 DOM 对象而非 zlib 对象

HTML 代码

<ul id="obj"><li>1</li><li title="new" >2</li><li>3</li><li>4</li></ul>

<ul><li>other</li></ul>

zLib 代码

$("#obj","li").one("title:new",function(el){

　　el.innerHTML = "zhenn";

});

or

$("#obj","li").one("title:new",function(el){

　　$(el).html("zhenn");

});

结果

<ul id="obj"><li>1</li><li title="new">zhenn</li><li>3</li><li>4</li></ul>
