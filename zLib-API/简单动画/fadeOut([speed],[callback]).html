fadeOut([speed],[callback])

通过设置元素不透明度来逐渐隐藏所匹配元素（只对非display:none元素有效）

参数

[speed](String)：为可选参数，渐隐的速度，有三个可选值，"slow","normal","fast"，默认值为"normal"
[callback](Function)：为可选参数，动画执行完毕时的回调函数

返回值

无

用法演示

单击按钮"fadeOut",逐渐减小元素的不透明度来隐藏元素，并在执行完毕后弹出提示信息！
