slideToggle([speed],[callback])

根据所匹配元素的显示状态选择执行slideDown或slideUp

参数

[speed](String)，为可选参数，滑动的速度，有三个可选值，"slow","normal","fast"，默认值为"normal"
[callback](Function)，为可选参数，动画执行完毕时的回调函数

返回值

无

用法演示

单击按钮"slideToggle",逐渐显示或逐渐隐藏元素，并在执行完毕后弹出提示信息！

HTML 代码

<style type="text/css">
*{ margin:0; padding:0;}

</style>
