toggle([speed],[callback])

根据所匹配元素的显示状态选择执行show或hide（无动画版本和动画版本）

注意：务必设置所匹配元素的width，不能为百分比并且不能使用内嵌式css(内联或外联均可)，否则会导致在display:none的情况下，获取的getWidth()不准确

参数

[speed](String)：为可选参数，变化的速度，有三个可选值，"slow","normal","fast"，默认值为"normal"
[callback](Function)：为可选参数，动画执行完毕时的回调函数

返回值

无

用法演示

单击按钮"toggle",通过逐渐改变元素的宽度、高度、不透明度来显示或隐藏元素，并在执行完毕后弹出提示信息！

HTML 代码

<style type="text/css">
*{ margin:0; padding:0;}
#obj{ color:red; border:1px solid #000; width:500px;}
#obj li{ background:#666;}
</style>

<ul id="obj">
　　 <li>zhenn</li>
　　 <li>do you known the eventListener?</li>
　　 <li>let me tell you </li>
　　 <li>success</li>
</ul>
<input type="button" value="toggle" id="toggle" />

zLib 代码

　　$("#obj").toggle("fast",function(){
　　　　alert("over");
　　});
});
