zlib.static.getType(obj)

准确判断对象（数据）类型，相对于typeof，增强了null,object,array的区分

参数

obj(任何类型)

返回值

String,如"string","function","array","object","null","undefined","number","regexp","math","date"

用法演示

检测数据类型

HTML 代码

无

zLib 代码

zlib.static.getType(function(){...}) //返回"function"

zlib.static.getType("test") //返回"string"

zlib.static.getType(1) //返回"number"

zlib.static.getType(null) //返回"null"

zlib.static.getType(undefined) //返回"undefined"

zlib.static.getType([1,2,3]) //返回"array"

zlib.static.getType({a:1,b:"zhenn"}) //返回"object"

zlib.static.getType(new Date()) //返回"date"

zlib.static.getType(Math) //返回"math"

zlib.static.getType(new RegExp()) //返回"regexp"
